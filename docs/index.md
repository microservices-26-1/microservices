# Projeto de Microservices

## Objetivo

Sistema baseado em microserviços desenvolvido para a disciplina de Platforms, Microservices, DevOps and APIs.

## Tecnologias

- Java
- Spring Boot
- Docker
- Docker Compose
- Gateway API

## Arquitetura

O sistema é composto por múltiplos microserviços independentes:

- Auth Service
- Account Service
- Product Service
- Order Service
- Gateway Service
